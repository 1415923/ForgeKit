## 回灌项分类

| 类型 | 说明 | 示例 |
| --- | --- | --- |
| 模板缺陷 | 生成项目自带内容会误导 Codex 或用户 | 保留 ForgeKit 历史任务 |

| 工具缺陷 | 脚本、自检、plugin 分发有问题 | 未检查中文 README |
| 项目特定经验 | 不应进入通用模板，只记录在外部试用记录 | 某个业务的特殊目录结构 |
