# Trial Feedback

Use this plugin to collect the 2-3 real project trials required by ForgeKit MAT-005.

For each trial, record:

- Project type and stack.
- Whether the plugin installed or loaded successfully.
- Whether initialization into the target project succeeded.
- Which skills were actually useful.
- Which instructions caused confusion or extra context load.
- Which validation commands passed or failed.
- Any missing local tools or setup tasks.
- Whether the project was Suitable, Conditional, or Custom.

Record trial summaries in the external ForgeKit project records first. Only promote stable learnings back into the template or plugin after review.
