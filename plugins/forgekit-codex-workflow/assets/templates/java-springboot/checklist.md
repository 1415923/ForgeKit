# Java Spring Boot 检查清单

## 开发前

- [ ] 明确接口、业务规则、数据库影响。
- [ ] 明确是否涉及 MySQL、Redis、Kafka、MinIO、Elasticsearch、Ollama / DeepSeek。
- [ ] 明确是否需要环境变量或配置项。

## 开发中

- [ ] Controller、Service、Repository 职责清晰。
- [ ] 输入参数有校验。
- [ ] 事务边界明确。
- [ ] 异常处理符合项目约定。
- [ ] 不硬编码密钥、密码、服务地址。

## 开发后

- [ ] 运行相关单元测试。
- [ ] 涉及 API 时更新接口文档。
- [ ] 涉及数据库时更新数据库设计或迁移说明。
- [ ] 涉及配置时更新部署文档。
- [ ] 涉及用户可见行为时更新版本记录。
