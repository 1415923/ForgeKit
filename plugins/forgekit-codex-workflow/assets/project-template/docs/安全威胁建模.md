## 适用场景

- 登录、鉴权、权限、角色变化。
- 用户数据或敏感业务数据处理。
- 文件上传、下载、解析。
- 外部 API、Webhook、MCP、第三方服务。
- 数据库查询、搜索、脚本执行。
- 部署和网络暴露变化。
