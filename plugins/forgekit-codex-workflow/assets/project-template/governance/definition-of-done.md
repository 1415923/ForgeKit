# Definition of Done

本文件定义“什么工作算完成”。

## 功能开发 Done

- [ ] 需求实现完成。
- [ ] 验收标准满足。
- [ ] 相关测试通过或说明无法运行原因。
- [ ] 接口、数据库、部署、配置文档已同步。
- [ ] 版本更新记录已更新。
- [ ] 追踪矩阵已更新。
- [ ] 项目任务看板状态已更新。
- [ ] 没有无关重构或无关文件混入。
- [ ] 安全和兼容影响已检查。

## Bug 修复 Done

- [ ] 问题已修复。
- [ ] 复现路径已验证。
- [ ] 增加回归测试或替代验证。
- [ ] 没有改变不该改变的外部行为。
- [ ] 缺陷修复计划和版本记录已更新。
- [ ] Bug / Task 状态已更新。

## Review/Refactor 中版本 Done

- [ ] 重复代码和重复文件已检查。
- [ ] 模块边界和目录结构已检查。
- [ ] 技术债已记录或处理。
- [ ] ADR/RFC 状态已同步。
- [ ] 文档已同步。
- [ ] 版本闸门结论已写明。
- [ ] Review/refactor 任务状态已更新。
- [ ] 是否允许进入下一个大版本已有结论。

## 发布 Done

- [ ] 发布检查通过。
- [ ] 版本路线图和版本记录一致。
- [ ] 回滚或恢复方案明确。
- [ ] 不包含密钥、调试文件、临时文件。
- [ ] 用户确认发布动作。
