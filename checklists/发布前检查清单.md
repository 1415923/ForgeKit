# 发布前检查清单

## 代码

- [ ] 满足 `governance/definition-of-done.md` 中的发布 Done。
- [ ] 工作区状态已检查
- [ ] 本版本改动已审查
- [ ] 无调试代码、临时文件、硬编码敏感信息
- [ ] 兼容性影响已评估
- [ ] 高影响变更已填写 `docs/change-impact.md`
- [ ] 回滚或规避方案已记录
- [ ] 严重事故或重复缺陷已复盘
- [ ] S2/S3 安全风险已关闭或人工接受
- [ ] 依赖安全审查已完成，若适用
- [ ] 环境矩阵已同步，目标环境明确
- [ ] 发布流水线已同步，制品和回滚路径明确
- [ ] Critical 或 Unknown 代码区域变更已有负责人或评审结论
- [ ] 当前版本任务状态已闭合，未完成项已有延期、裁剪、阻塞或放弃结论

## 测试

- [ ] 单元测试通过
- [ ] 集成测试通过或说明不适用
- [ ] 端到端测试通过或说明不适用
- [ ] 构建通过
- [ ] 静态检查通过或说明剩余问题

## 文档

- [ ] 需求文档已同步
- [ ] 架构文档已同步
- [ ] 接口文档已同步
- [ ] 数据库设计或迁移说明已同步
- [ ] 部署文档已同步
- [ ] 环境矩阵已同步
- [ ] 发布流水线已同步
- [ ] 代码所有权已同步
- [ ] 项目任务看板已同步
- [ ] 版本更新记录已同步
- [ ] 变更影响评估已同步
- [ ] 事故复盘或缺陷复盘已同步，若适用
- [ ] 安全威胁建模和依赖安全审查已同步，若适用

## 发布

- [ ] 版本号已确定
- [ ] Tag 规则已确认
- [ ] 回滚方案已确认
- [ ] 部署步骤已确认
- [ ] 生产发布、数据库迁移、Git tag、镜像或包发布等动作已获得人工确认
- [ ] 发布风险已确认
