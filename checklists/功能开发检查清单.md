# 功能开发检查清单

## 开发前

- [ ] 满足 `governance/definition-of-ready.md`。
- [ ] 当前工作已有 EPIC / FEAT / TASK / BUG 编号
- [ ] 当前工作在 `docs/项目任务看板.md` 中处于 Ready / Todo
- [ ] 需求和验收标准明确
- [ ] 当前范围属于 `.codex/scope.md`
- [ ] 已识别影响文件
- [ ] 已对照 `docs/代码所有权.md` 确认影响路径的所有者和必要评审人
- [ ] 已确认是否需要更新接口、数据库或配置
- [ ] 已确认是否涉及安全、权限、凭据、外部服务或 MCP
- [ ] 若涉及 API、数据库、配置、部署、权限、安全、外部服务或硬件接口，已更新 `docs/变更影响评估.md`
- [ ] 若涉及 S2/S3 安全敏感变更，已更新 `docs/安全威胁建模.md`
- [ ] 若新增或升级依赖，已更新 `docs/依赖安全审查.md`
- [ ] 若命中 Critical 或 Unknown 代码区域，已获得负责人或人工确认
- [ ] 复杂任务已有计划，高风险核心逻辑已考虑是否需要 TDD

## 开发中

- [ ] 遵循项目代码风格
- [ ] 修改范围与任务相关
- [ ] 没有引入无关重构
- [ ] 错误处理和日志符合项目约定
- [ ] 敏感信息没有写入代码或文档

## 开发后

- [ ] 满足 `governance/definition-of-done.md`。
- [ ] 运行相关测试
- [ ] 运行必要构建或静态检查
- [ ] 安全敏感变更已按 `.codex/security.md` 检查
- [ ] S2/S3 安全风险已关闭或人工接受
- [ ] 高影响变更已完成影响评估、验证和回滚说明
- [ ] Critical 或 Unknown 代码区域变更已有评审结论
- [ ] 更新相关文档
- [ ] 更新 `docs/项目任务看板.md`
- [ ] 更新版本更新记录
- [ ] 查看 `git diff`
- [ ] 确认没有混入无关改动
