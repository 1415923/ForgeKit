# Go Service 命令示例

```powershell
go version
go mod tidy
go test ./...
go test -race ./...
go vet ./...
go fmt ./...
go build ./...
```

## Local validation / 局部验证优先级

```powershell
go fmt ./...
go vet ./...
go test ./...
go build ./...
```

注意：

- `go mod tidy` 会修改 `go.mod` / `go.sum`，先确认。
- `go test -race ./...` 更慢，适合并发、网络、共享状态变更后运行。
- 启动服务、访问真实数据库、发布二进制或容器镜像前必须确认。
