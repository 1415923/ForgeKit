# 初始化填充 Prompt

请使用 `.agents/skills/project-bootstrap-fill`，根据项目初始化问答内容补齐项目第一版规则和文档。

要求：

1. 读取 `governance/project-bootstrap-fill.md`、`governance/overview.md`、`.codex/init.generated.md`、`.codex/questionnaires/init-questionnaire.md`、`.codex/stacks/` 中相关技术栈模板。
2. 从问答中提取稳定事实，不要编造未知信息。
3. 将事实合并到以下文件：
   - `.codex/project.md`
   - `.codex/scope.md`
   - `.codex/commands.md`
   - `.codex/style.md`
   - `.codex/testing.md`
   - `.codex/security.md`
   - `.codex/git.md`
   - `.codex/version-gates.md`
   - `docs/project-plan.md`
   - `docs/tech-decisions.md`
   - `docs/version-roadmap.md`
   - `docs/requirements.md`
   - `docs/environment-matrix.md`
   - `docs/release-pipeline.md`
   - `docs/code-ownership.md`
   - `docs/task-board.md`
   - `docs/traceability.md`
   - `docs/risk-register.md`
4. 如果信息不足，保留 `待确认`，并提出 3-5 个最关键方案追问。
5. 不要修改业务代码。

输出格式：

- 已填充文件
- 已确认事实
- 待确认问题
- 下一轮方案访谈问题
- 初始任务拆分
- 风险和假设
- 是否允许进入编码
