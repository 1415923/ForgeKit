## 审查问题

- 为什么需要这个依赖？
- 是否已有项目内替代？
- 是否有更轻量或更可信替代？
- 是否会引入网络、文件、权限、执行代码风险？
- 是否影响构建、部署、许可证？
- 是否需要固定版本？
- 是否有已知漏洞？
- 是否会改变 lockfile 或构建产物？
