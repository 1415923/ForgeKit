# 风险登记册

## 状态

| 状态 | 含义 |
| --- | --- |
| Open | 已识别，未处理 |
| Mitigating | 正在缓解 |
| Accepted | 接受风险 |
| Closed | 已关闭 |

## 使用规则

- 高影响或高概率风险必须有缓解措施。
- 影响版本推进的风险必须关联版本路线图。
- 架构风险应关联 RFC 或 ADR。
- 接手项目的兼容风险必须记录。
