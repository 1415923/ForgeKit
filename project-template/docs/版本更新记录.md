### Added

- 新增 `governance/agent-harness.md`，定义 Codex 上下文加载、agentic search、AGENTS 分层和停止编码条件。
- 新增 `governance/large-change-execution.md`，定义大任务探索、计划、分会话执行和 review 闸门。
- 新增 `docs/代码库地图.md`，作为项目模块、入口文件、搜索关键词和局部验证命令的维护入口。
- 新增 `docs/本地工具链检查.md`，记录 LSP、lint、test、build 和局部验证能力。
- 新增 `docs/探索报告.md` 和 `docs/实施计划.md`，用于跨模块或高风险改动前的只读探索和阶段计划。
- 新增 `governance/team-agent-rollout.md`，定义 commands、hooks、plugin、MCP 和团队推广顺序。
- 新增 `.codex/commands-catalog.md` 和 `.codex/hooks.md`，记录可选命令、GitHub 风格集成候选和 hook 示例。
- 新增 `governance/agent-suitability.md`、`docs/项目适用性评估.md`、`docs/真实项目试用记录.md`，建立项目适用性评估和真实项目回灌机制。
- 新增 `scripts/detect-local-toolchain.ps1`，用于只读检测本地工具链和 LSP 候选。
- 新增 `scripts/run-harness-check.ps1`，用于生成项目内检查 harness 入口是否完整。
- 新增 `docs/Codex下一步工作单.md`，用于初始化后继续访谈和收敛方案。
