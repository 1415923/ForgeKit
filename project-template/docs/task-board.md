# 项目任务看板

用于记录当前项目的 Epic、Feature、Task、Bug、Test 状态和版本映射。不要把 ForgeKit 模板自身的历史任务写入生成项目。

任务看板记录拆解后的执行状态，不替代 `.forgekit/docs/task-intake.md` 中的任务派发原文台账。来自领导任务、微信任务、计划表格子、会议任务或手工记录的条目，必须能通过 Source ID 反链到 `.forgekit/docs/task-intake.md`。

## 状态定义

| 状态 | 说明 |
| --- | --- |
| Backlog | 已记录，未准备开发 |
| Ready | 已满足 Definition of Ready |
| Todo | 已排入当前版本，尚未开始 |
| In Progress | 正在实现 |
| Review | 等待审查或验收 |
| Done | 已满足 Definition of Done |
| Blocked | 被阻塞，已记录原因 |
| Dropped | 已裁剪或放弃，已记录原因 |
