# Archived Releases

Store old release bundles, release notes, verification evidence, or handoff material here when they are no longer current.

Suggested layout:

```text
.forgekit/archive/releases/
  YYYY/
    vX.Y.Z/
```

Archived releases are historical evidence. Current release state belongs in `.forgekit/docs/changelog.md`, `.forgekit/docs/version-roadmap.md`, and the active release change folder when one exists.
